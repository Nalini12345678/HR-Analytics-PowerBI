## Project Overview
This project presents a complete HR Analytics Dashboard built using Power BI.
The goal is to analyze employee attrition, workforce trends, and department-level performance.

## Dataset
1200 Employee Records
Includes Salary, Age, Gender, Department, Hire Date, Exit Date
Data cleaned using Power Query

## Data Cleaning Steps
Handled null values in Salary and Age
Created Status column (Active / Exited)
Created Tenure column
Fixed gender inconsistencies
Validated date logic
Removed duplicates

## DAX Measures Used

Active Employees
Employees Left
Attrition Rate %
Average Salary
Average Tenure
